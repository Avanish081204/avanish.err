# My Digital Showcase

This is the source code for the personal portfolio of **Avanish Shukla**, a full‑stack web developer.

## Getting started

Make sure you have Node.js and npm installed.

```sh
npm install
npm run dev
```

The app will start on `http://localhost:8080` (or the port shown in your terminal).

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Building for production

```sh
npm run build
```

You can then deploy the generated `dist` folder to any static hosting provider (Netlify, Vercel, GitHub Pages, etc.).
